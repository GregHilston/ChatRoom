ChatRoom
========

Java chat room, written to refamiliarize myself with basic thread operations and object orientated design. Also, will ideally use this chat room is other projects down the road


### Features
========
* Socket based server
* Threaded
* Event and chat logging
* Private messages
* Separate channels
* Server commands
